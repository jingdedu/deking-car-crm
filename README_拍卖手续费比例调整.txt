V4.1 报价单拍卖手续费调整

计算规则：
1. 基础拍卖手续费 = 预计成交价 × 可调比例
2. 可调比例范围：2.0% ～ 2.8%，步进 0.1%，默认 2.2%
3. 当预计成交价 > 2,850万韩元时，再追加：预计成交价 × 7% × 15%
4. 拍卖手续费合计 = 基础拍卖手续费 + 高价追加手续费

示例：
预计成交价 2,500万，比例 2.2%：手续费 = 55万
预计成交价 3,000万，比例 2.2%：基础66万 + 追加31.5万 = 97.5万（显示时四舍五入）
